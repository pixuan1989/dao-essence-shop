## 常见问题排查

### 问题 1: 购物车仍然显示 0.00

**可能原因**：
1. ❌ registerCreemProducts 未被调用
2. ❌ creem-sync-v2.js 加载失败
3. ❌ 浏览器缓存未清空

**排查步骤**：
```bash
1. 打开 F12 Console
2. 运行命令：console.log(window.products)
   应该看到 8 个产品（6 个硬编码 + 2 个 Creem）
3. 运行命令：console.log(window.cart.items)
   应该看到添加的产品
```

### 问题 2: 图片未加载

**可能原因**：
1. ❌ img_url 字段为空
2. ❌ 图片 URL 返回 404
3. ❌ CORS 问题

**排查步骤**：
```bash
1. 打开 F12 Network 标签
2. 查看所有 GET 请求
3. 查找 /images/ 或 CDN URL
4. 检查返回状态是否为 200
5. 如果 404，检查 img_url 值是否正确
```

### 问题 3: 刷新后购物车清空

**可能原因**：
1. ❌ localStorage 已禁用
2. ❌ 浏览器隐私模式
3. ❌ saveCartToStorage() 未被调用

**排查步骤**：
```bash
1. F12 Console 运行：
   localStorage.setItem('test', 'value')
   localStorage.getItem('test')
   
2. 应该看到返回 "value"
3. 如果返回 null 或错误，localStorage 可能被禁用
```
