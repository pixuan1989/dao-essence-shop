# DaoEssence 首页重设计思路文档

## 一、问题诊断

### 当前首页的问题
1. **价值主张模糊** — 首屏没有一句话说清楚"我们是做什么的"
2. **缺乏信任锚点** — 没有用户评价、案例、或权威背书
3. **行动路径不清晰** — CTA（Call to Action）分散，用户不知道下一步该做什么
4. **内容密度过高** — 五行环动画很酷，但信息过载，反而分散注意力

### 对比 livelearnbazi.com 的优势
| livelearnbazi.com | DaoEssence 现状 |
|:---|:---|
| 首屏大字标题："Learn BaZi" + 副标题解释 | 首屏只有五行动画，没有文字说明 |
| 清晰的3步流程：Learn → Practice → Master | 没有明确的用户旅程 |
| 免费工具入口（Free BaZi Calculator）作为引流 | 没有免费钩子 |
| 用户评价 + 成功案例建立信任 | 没有社会证明 |
| 简洁的导航：Courses / Tools / Blog / About | 导航项目过多 |

---

## 二、设计原则（保留现有风格）

- ✅ 极简主义、留白、衬线字体
- ✅ 五行环/太极动画（但位置调整）
- ✅ 黑白灰主色调 + 金色点缀
- ✅ 宽间距、大字体

---

## 三、重设计思路

### 1. 首屏 Hero Section（完全重构）

**当前**：五行动画占满全屏，没有文字

**改为**：
```
[左侧 60%]                    [右侧 40%]
大字标题（一句话价值主张）     五行环动画（缩小，作为视觉点缀）
副标题（解释我们是做什么的）
主 CTA 按钮（免费八字排盘 / 了解五行）
次 CTA 链接（查看商城产品）
```

**文案建议**：
- 主标题："读懂你的命盘，掌握人生方向"
- 副标题："专业八字命理分析 · 五行能量调和 · 东方智慧现代解读"
- CTA："免费排盘" / "了解更多"

---

### 2. 信任背书 Section（新增）

**位置**：首屏下方

**内容**：
- 3 个数字指标："1000+ 命盘分析" / "98% 用户好评" / "50+ 五行产品"
- 或 3 条用户评价（可先用占位符）

---

### 3. 服务/产品矩阵 Section（重构现有内容）

**当前**：五行介绍 + 数字产品 + 物理产品混在一起

**改为**：清晰的 3 列卡片
```
[命理咨询]              [五行文化]              [能量产品]
八字排盘分析            五行理论详解            水晶/饰品/香薰
免费在线工具            入门学习指南            根据命局推荐
→ 立即排盘              → 开始学习              → 浏览商城
```

---

### 4. 免费钩子 Section（新增）

**学习 livelearnbazi.com 的 Free BaZi Calculator**

**内容**：
- 标题："免费八字排盘"
- 副标题："输入出生时间，获取你的五行命盘分析"
- 简化的输入框（年份/月份/日期/时辰）
- 提交按钮
- 说明："完全免费，无需注册"

**目的**：降低用户门槛，获取首单转化

---

### 5. 内容/博客 Section（预留）

博客功能已在 feature/blog-cms 分支开发，首页预留入口：
- "命理知识" 板块
- 展示最新 3 篇文章
- 链接到 /blog/

---

### 6. Footer 简化

**当前**：信息过多
**改为**：简洁的 4 列布局
- 品牌 + 简介
- 快速链接
- 联系方式
- 社交媒体

## 四、信息架构对比

### 当前首页结构
```
Hero（只有动画）
↓
五行介绍（木火土金水）
↓
核心价值观（3列）
↓
数字产品
↓
BaZi 排盘服务
↓
物理产品
↓
联系/页脚
```

### 建议首页结构
```
Hero（价值主张 + 动画点缀）
↓
信任背书（数字/评价）
↓
服务矩阵（3列卡片）
↓
免费钩子（八字排盘入口）
↓
精选内容/博客（预留）
↓
Footer
```

---

## 五、技术实现要点

1. **响应式**：移动端单列堆叠，桌面端左右分栏
2. **动画调整**：五行环缩小到 40% 宽度，不作为绝对主角
3. **性能**：首屏加载优先文字内容，动画延迟加载
4. **SEO**：首屏 H1 标题包含核心关键词
