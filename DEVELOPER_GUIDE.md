# DAO Essence 开发人员指南

## 项目概述

DAO Essence 是一个垂直精细化的电商网站，专注于道家文化相关产品的销售。本指南提供了项目的功能说明、技术架构和开发指南，帮助开发人员快速理解和维护项目。

## 核心功能

### 1. 产品管理
- **产品展示**：响应式产品列表和详情页
- **产品分类**：支持按类别浏览产品
- **产品详情**：包含产品描述、价格、库存等信息
- **产品数据**：通过 `products.json` 文件管理产品数据

### 2. 购物车功能
- **添加商品**：支持将商品添加到购物车
- **购物车管理**：修改商品数量、移除商品
- **购物车持久化**：使用 localStorage 保存购物车数据
- **购物车计算**：自动计算总价和数量

### 3. 支付系统
- **Creem集成**：支持全球支付，特别适合欧美客户
  - Creem Checkout 模式
  - 支持信用卡、借记卡等多种支付方式
- **测试模式**：支持使用Creem测试模式
- **生产模式**：支持真实支付
- **支付验证**：通过 Webhook 验证支付状态

### 4. 订单管理
- **订单创建**：支付成功后自动创建订单
- **订单通知**：发送邮件通知给买家和商家（作为主要订单管理方式）
- **订单状态**：通过邮件记录跟踪订单状态
- **订单历史**：通过邮件历史记录查看订单历史
- **物流集成**：接入货运平台，自动获取物流单号并发送通知

### 5. 邮件通知
- **阿里云邮件集成**：使用阿里云邮件推送服务
- **买家确认邮件**：包含订单详情、支付信息和预计发货时间
- **商家通知邮件**：包含订单详情和客户信息
- **发货通知邮件**：包含订单号、发货时间、物流单号、物流公司名称、物流查询链接和预计送达时间
- **邮件模板**：美观的 HTML 邮件模板

#### 邮件必须包含的内容

**买家确认邮件**：
- 订单号和下单时间
- 支付金额和支付方式
- 商品清单（名称、数量、价格）
- 预计发货时间（3-5个工作日）
- 收货地址（如果提供）
- 联系信息和客服联系方式

**商家通知邮件**：
- 订单号和下单时间
- 客户姓名和联系方式
- 支付金额和支付方式
- 商品清单（名称、数量、价格）
- 收货地址（如果提供）
- 待办提醒（3-5个工作日内发货）

**发货通知邮件**：
- 订单号和发货时间
- 物流公司名称
- 物流单号
- 物流查询链接
- 商品清单（名称、数量）
- 预计送达时间（15-25个工作日）
- 联系信息和客服联系方式

#### 物流集成逻辑
1. **接入货运平台**：与物流服务商API集成
2. **自动获取单号**：货运平台发货后自动生成物流单号
3. **数据同步**：物流单号自动推送到平台系统
4. **自动通知**：系统自动发送发货通知邮件给买家
5. **客服通知**：同时通知客服已发货，便于客服跟进
6. **物流跟踪**：邮件中包含物流查询链接，方便用户实时跟踪

### 6. 网站管理
- **管理面板**：后台管理功能，支持商品的添加、编辑、删除和管理
- **商品管理**：完整的商品CRUD操作，包括库存管理和价格调整
- **数据存储**：使用MongoDB Atlas云数据库存储商品数据
- **配置管理**：通过环境变量配置系统
- **日志管理**：系统运行日志和错误记录

## Vercel 迁移计划

### 迁移原因
- **Netlify 信用额度限制**：团队版容易超出信用额度导致部署暂停
- **Vercel 个人版永久免费**：没有信用额度限制，更稳定
- **更好的性能**：Vercel 的边缘网络和构建速度更优
- **开发体验**：Vercel 对前端框架的支持更好

### 迁移步骤
1. **准备工作**：
   - 确保代码在 GitHub 仓库中
   - 备份所有环境变量配置
   - 记录当前的部署配置

2. **创建 Vercel 项目**：
   - 登录 Vercel 账户
   - 点击 "New Project"
   - 选择 GitHub 仓库
   - 配置构建设置

3. **迁移函数**：
   - 将 Netlify Functions 转换为 Vercel Functions
   - 创建 `/api` 目录结构
   - 适配函数格式和配置

4. **配置环境变量**：
   - 在 Vercel Dashboard 中设置所有环境变量
   - 包括 Creem 凭证、阿里云配置等

5. **测试**：
   - 完成支付流程测试
   - 测试邮件发送功能
   - 验证 Webhook 处理

6. **DNS 配置**：
   - 更新域名 DNS 记录，指向 Vercel
   - 等待 DNS 记录生效
   - 启用 SSL 证书

7. **切换**：
   - 确认 Vercel 部署正常
   - 将流量切换到 Vercel
   - 监控一段时间确保稳定

### 迁移注意事项
- **函数兼容性**：Netlify Functions 和 Vercel Functions 有细微差异
- **环境变量**：需要重新配置所有环境变量
- **Webhook URL**：需要更新 Creem Webhook 端点
- **DNS 传播**：可能需要 24-48 小时
- **数据备份**：迁移前备份所有重要数据

### 优先级：高
- 迁移到 Vercel 是当前的重要任务
- 避免 Netlify 信用额度问题导致的部署中断
- 提高网站性能和稳定性

## Creem 集成指南

### 1. Creem 账户注册
1. 访问 [Creem 官网](https://www.creem.io/)
2. 注册 Creem 账户
3. 创建测试模式账户
4. 获取 API 凭证（API Key 和 Secret Key）

### 2. Creem Webhook 配置
1. 在 Creem Dashboard 中进入 Webhook 设置
2. 添加 Webhook 端点：`https://your-domain.com/api/webhooks/creem`
3. 选择需要监听的事件：
   - `checkout.completed`
   - `payment.failed`
4. 获取 Webhook Secret

### 3. Creem 集成特点
- **优势**：
  - 全球支付平台，支持多种支付方式
  - 支持多国家和地区，多币种
  - 支持信用卡、借记卡等多种支付方式
  - 易于集成，API 文档完善
  - 可靠的支付处理和安全保障
- **费用**：交易费合理，具体请参考 Creem 官方文档
- **适用场景**：所有类型的跨境电商，特别适合欧美客户
