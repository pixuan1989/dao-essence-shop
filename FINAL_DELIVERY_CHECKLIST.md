## ⚠️ 常见问题和解决方案

### 问题 1：购物车仍然显示 0.00

**原因**：registerCreemProducts() 未被调用

**解决**：
```bash
# 1. 清空浏览器缓存和 localStorage
F12 → Application → Clear site data

# 2. 查看 Console 是否有错误
# 3. 检查 /api/products 是否返回 200
# 4. 如果是生产环境，检查 Vercel 部署是否完成
```

### 问题 2：Console 中有 Creem 错误

**可能原因**：
1. API 服务暂时不可用
2. 环境变量未设置
3. 网络错误

**检查方式**：
```bash
# 1. 打开 F12 Network 标签
# 2. 查找 /api/products 请求
# 3. 查看响应状态和错误消息
# 4. 检查 localStorage 缓存是否存在
```

### 问题 3：图片不显示

**原因**：img_url 为空或返回 404

**排查**：
```bash
F12 → Network → 查找所有 .jpg/.png 请求
查看是否都返回 200
如果 404，检查 Creem API 返回的 img_url 值
```
