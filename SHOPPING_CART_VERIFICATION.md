- **浏览器**：Chrome, Firefox, Safari 等任何现代浏览器
- **Developer Tools**：F12 打开 Console 查看调试日志

### 步骤1：打开页面
1. 打开上方 URL
2. 页面应加载完成，右下角出现金色购物车按钮
3. 购物车徽章显示 **0**

### 步骤2：添加商品
1. 页面上方找到商品数量选择器（左边 - 符号，中间数字，右边 + 符号）
2. 点击 + 几次，改变数量为 **3**
3. 点击黄色的 "加入购物车" 按钮
4. **预期结果**：
   - [ ] 右下角徽章变为 **3**
   - [ ] 屏幕顶部出现通知：`"Added 3 item(s) to cart"`
   - [ ] 购物车侧边栏自动打开

### 步骤3：查看购物车
1. 侧边栏已打开
2. **检查内容**：
   - [ ] 顶部有标题 "Shopping Cart"
   - [ ] 顶部右角有 × 关闭按钮

   - [ ] 底部显示 Total: $XX.XX
   - [ ] 底部有 "Checkout →" 按钮

### 步骤4：关闭购物车（测试修复1）
**方式1：点击关闭按钮**
1. 点击侧边栏头部的 × 按钮
2. **预期结果**：侧边栏滑出，黑色覆盖层消失 ✓

**方式2：点击覆盖层**
1. 重新点击购物车按钮打开
2. 点击黑色区域（侧边栏左侧）
3. **预期结果**：侧边栏滑出，黑色覆盖层消失 ✓

**方式3：再次点击购物车按钮**
1. 重新点击购物车按钮
2. **预期结果**：侧边栏切换状态，关闭 ✓

### 步骤5：再次添加商品（测试数量累加）

## 🔍 调试技巧

### 打开 Browser Console
- **快捷键**：F12（大多数浏览器）
- **或**：右键 → 检查 (Inspect)

### 查看调试日志
1. 点击购物车按钮 → Console 应显示（若有日志）
2. 添加商品 → 查看 `addToCart()` 输出
3. 关闭购物车 → 查看相关函数调用

### 常见日志
```javascript

// 如果有错误
```
